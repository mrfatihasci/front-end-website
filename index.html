Branching

    1. git branch ------- // see current branches.
    2. git branch branch_name ---------- // create a develop branch named branch_name.
    3. git checkout branch_name --------- // change(checkout) to branch_name.
    4. git checkout -b branch_name -------- // use #2 and #3 in a single command.
    5. git push origin branch_name -------- // push branch & now u can see this branch in GitHub repo(after add & commit).

    1. git merge branch_name -------- // take all commited changes in branch_name & add to the branch you are currently on.
    2. git push origin main -------- // push branch(main / master ).

    1. git branch -d branch_name ----------- // delete branch_name from your local repo if it is merged.
    2. git branch -D branch_name ------------ // delete branch_name from your local repo if it is not merged.
    3. git push origin --delete branch_name ---------- // delete branch_name from your remote repo.
