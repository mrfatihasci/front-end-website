GitHub

Refer to this book for any further git and github related questions.

  1. Setup
    • sudo add-apt-repository ppa:git-core/ppa
    • sudo apt update
    • sudo apt install git
    • git --version // see if installed

  2. Settings
    • git config --global user.name "Your Name"
    • git config --global user.email "yourname@example.com"
    • git config --global init.defaultBranch main //setting default branch as main since new repos in github has default branch as main instead of master
    • Next, google how to create ssh key, how to link ssh key to your github account by yourself

  3. Working in a repo

    ...


  4. Branching
    • creating & pushing branch
      • git branch ------- // see current branches.
      • git branch branch_name ---------- // create a develop branch named branch_name.
      • git checkout branch_name --------- // change(checkout) to branch_name.
      • git checkout -b branch_name -------- // use #2 and #3 in a single command.
      • git push origin branch_name -------- // push branch & now u can see this branch in GitHub repo(after add & commit).

    • merging branches
      • git merge branch_name -------- // take all commited changes in branch_name & add to the branch you are currently on.
      • git push origin main -------- // push branch(main / master ).

    • deleting unnecessary merged/unmerged branches
      • git branch -d branch_name ----------- // delete branch_name from your local repo if it is merged.
      • git branch -D branch_name ------------ // delete branch_name from your local repo if it is not merged.
      • git push origin --delete branch_name ---------- // delete branch_name from your remote repo.

  5. Undo Add/Commit when not pushed
    1. git reset --soft HEAD1 ----------- // does uncommit the last 1 commit yet does not unstage (u can unstage changes by following "git status"s instruction).
    2. git reset --mixed HEAD1 ------------ // ..
    3. git reset --hard HEAD1 ---------- // ..
